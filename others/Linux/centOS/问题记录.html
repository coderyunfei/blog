              # 问题记录

              # xxx is not in the sudoers file. This incident will be reported

              表明 xxx 用户没有权限是 sudo 命令，将该用户加入 /etc/sudoers 即可：

              1. 切换到超级管理员：su -
              2. 添加文件写权限：chmod u+w /etc/sudoers
              3. 使用 vim 打开文件：chmod u+w /etc/sudoers
              4. 按 ↓ 移到光标大概到 106 行，会看到 root ALL=(ALL) ALL
              5. 光标在 root ALL=(ALL) ALL 头部，按 y 复制该行
              6. 光标移到下一行，按 p，粘贴
              7. 按 i 进入编辑模式，修改 root 为 xxx
              8. 按 esc 或 v 进入可视模式
              9. :wq，保存文件，退出
              10. 撤销文件的写权限：chmod u-w /etc/sudoers

              切换到其他用户 su use-name
